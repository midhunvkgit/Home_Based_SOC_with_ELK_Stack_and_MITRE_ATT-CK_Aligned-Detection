# Home_Based_SOC_with_ELK_Stack_and_MITRE_ATT-CK_Aligned-Detection
Home-based SOC built with ELK Stack and Beats, featuring MITRE ATT&amp;CK-aligned detection logic and Kibana dashboards for threat visualization.

This project presents the design and deployment of a home-based Security Operations Center (SOC) that leverages open-source technologies for real-time threat detection, log analysis, and adversary behavior mapping. The core of the architecture is the ELK Stack—Elasticsearch, Logstash, and Kibana—used for data ingestion, processing, storage, and visualization. Endpoint telemetry is collected using Filebeat for Linux logs, Winlogbeat for Windows Event Logs, and Auditbeat to monitor file integrity, user activity, and process execution on Linux systems, enhancing system-level visibility. These agents forward data to Logstash, which parses and enriches the events before indexing them into Elasticsearch. Elasticsearch functions as the centralized data store and analytics engine, indexing and managing log data at scale while enabling fast, full-text search, aggregations, and time-series analytics across diverse data sources, forming the analytical backbone of the SOC. 

The project incorporates adversarial simulation using Atomic Red Team and Metasploit to emulate real-world attack techniques, allowing the testing of existing detection capabilities. Once the simulations are complete, custom detection rules are created based on the observed attack behaviors to identify specific threats and malicious activities. These rules are then implemented within Kibana to enable precise detection of abnormal patterns and behaviors across the collected data. The MITRE ATT&CK framework is utilized to map detected behaviors to known adversary tactics and techniques, promoting a threat-informed defense strategy. Kibana provides an intuitive interface for visual exploration, dashboard creation, and event correlation, supporting both investigative workflows and executive-level reporting. Real-time alerting is integrated via Kibana Alerts, with email notifications set up to inform administrators whenever predefined thresholds or threat patterns are met. This project demonstrates that with minimal infrastructure, a fully functional and extensible SOC environment can be built, making it suitable for cybersecurity training, research, and entry-level operational use.
